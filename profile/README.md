# Mythic Framework

> A modular, community-driven framework for CFX.re platforms (FiveM).

---

## 🚀 Getting Started

Mythic Framework now uses a **txAdmin Recipe** for streamlined setup and deployment.

All framework components are split into **modular repositories** under this organization to support scalability, flexibility, and community contributions.

👉 **Start here:**
🔗 https://github.com/Mythic-Framework/txAdminRecipe

### The recipe handles:

* Automatic cloning of required resources
* Proper resource structure
* txAdmin-ready configuration
* Easier updates and customization

🔍 Browse all modules:
https://github.com/orgs/Mythic-Framework/repositories

## 🗺️ Map Dependencies

Mythic Framework supports a variety of MLOs and map resources.

Some are included and started automatically, while others must be added manually.

⚠️ Certain legacy maps may no longer be available — you may replace them with alternatives and adjust coordinates accordingly as we process into free alternatives for ease of use
